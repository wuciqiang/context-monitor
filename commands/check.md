# 快速上下文检查

快速检查当前上下文使用率并提供建议。

## 执行步骤

1. **调用工具** (带降级方案)
   - **主方案**: 调用 `check_context_usage` MCP 工具
   - **降级方案**: 如 MCP 失败，读取 statusline 临时文件
   - 获取实时上下文使用率

2. **解读结果**
   - 显示使用率百分比
   - 显示状态（SAFE/WARNING/HIGH/CRITICAL）
   - 显示建议行动

3. **自动行动**（如果需要）
   - 如果 > 85%：自动调用 `save_session_state`
   - 如果 > 70%：提醒用户准备保存状态

**输出**：上下文状态报告 + 行动建议
